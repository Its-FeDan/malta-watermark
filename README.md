# Malta Watermark-o-Matic 3000

Small static web app that watermarks an uploaded ID/passport image with:

```text
For Sliema Central Apt. 8, Malta stay, July 2-5 2026
```

The app runs entirely in the browser. It has no backend and does not upload the chosen image.

## Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy

Import this repo in Vercel as a static project. No build command is required.
